## Propósito  
Este repositorio contiene el desarrollo de una aplicación de escritorio con acceso a base de datos, implementando arquitectura multicapas.

## Estructura  
El proyecto está organizado en:
- Capa de Presentación  
- Capa de Lógica de Negocio  
- Capa de Acceso a Datos
